# 📝 Kişisel Not Defteri Uygulaması

Bu Python tabanlı uygulama, kullanıcıların kişisel notlarını oluşturabileceği, güncelleyebileceği ve görüntüleyebileceği basit bir not defteri sistemidir.

## 🚀 Özellikler- Kullanıcı girişi (İsim kontrolü)
- Yeni not dosyası oluşturma
- Var olan notları güncelleme
- Notları görüntüleme
- Türkçe karakter desteği
- Basit ve kullanıcı dostu terminal arayüzü

## 🛠 Kullanılan Teknolojiler
- Python 3
- `os` modülü (dosya işlemleri için)
- `unicodedata` (unicode karakter işlemleri için, opsiyonel)

## 🧑‍💻 Kurulum ve Kullanım

1. **Projeyi indir / klonla:**

```bash
git clone https://github.com/kullaniciadi/not-defteri-uygulamasi.git
cd not-defteri-uygulamasi

2.Python ile çalıştır:

python not_defteri.py

3.Kullanıcı ismi kontrolü için:

İsimler.txt dosyasına geçerli isimleri satır satır yazmalısınız.
Kullanıcının girdiği isim bu listede yoksa giriş yapılmaz.

Notlar
Dosya adlarında Türkçe karakter kullanımı desteklenmektedir.
Aynı isimle ikinci kez dosya oluşturulamaz.
Zeynep'in Notu.txt, Yaz Stajı.txt gibi ön tanımlı dosyaları güncelleyebilir ya da görebilirsiniz.
